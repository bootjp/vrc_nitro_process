## what is this?（これはなに？）

VRChatのログフォルダを監視し，新たにログが生成されたらプロセスを立ち上げます．

リアルタイムでログをパースするアプリケーションの管理に用いることを想定しています．

## how to use （使い方）

`example.yml` を例に起動したいプロセスを `setting.yml` を設定します．

このプロジェクトの実行ファイルと同じ階層に `setting.yml` を設置します．
